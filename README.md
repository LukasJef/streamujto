# STREAMUJ.TO
## O čem je?
Využívá existující stránky, obchází reklamy, předplatná a přehrává surový soubor videa, přičemž používá přebaly z veřejných databází.
## Nějaké funkce:
Všechny údaje o přehrávání se ukáladjí lokálně včetně možnosti přidat film do oblíbených nebo ho stáhnout.
## Díky:
Velké díky patří Saros, protože udělal [tenhle](https://github.com/Saros72/kodirepo/tree/main/repo-19/plugin.video.prehrajto) plugin do kodi, podle kterého jsem tuhle stránku mohl udělat co nejlépe.
